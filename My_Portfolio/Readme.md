# Personal Portfolio Website

This is a simple one-page personal portfolio website created for the Panaversity “Markdown In, HTML Out” task.

## About the Project

The website was first planned using Markdown and then converted into a clean, mobile-friendly HTML portfolio page using AI.

## Sections Included

* Name / Main heading
* About Me
* Skills
* Projects
* Contact

## Tools Used

* Markdown
* ChatGPT
* HTML
* CSS
* GitHub

## Prompts Used

1. Turn this Markdown into a clean, mobile-friendly, one-page portfolio website in HTML.
2. Grade this page out of 10 for design and readability, then improve it.
3. Add a dark colour theme, project cards, contact links, and a small animation.
